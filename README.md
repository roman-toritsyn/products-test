# Product Store - Dynamic Theming Challenge

A Next.js application demonstrating server-side dynamic theming with Tailwind CSS, featuring a product store that randomly selects visual templates on each server request.

## 🎯 Task Overview

This project implements a product store page (`/store`) with dynamic server-side theming. The visual theme is randomly selected between two templates ("Classic Blue" and "Modern Emerald") on each request, demonstrating advanced Next.js architecture and Tailwind CSS configuration.

### Key Requirements Met

✅ **Dynamic Server-Side Configuration** - Random theme selection via API routes  
✅ **CSS Variables Integration** - Theme colors implemented as CSS variables  
✅ **Tailwind Configuration** - Custom colors mapped to CSS variables  
✅ **404 Page with Theming** - Error page inherits current theme colors  
✅ **Pixel-Perfect Design** - 1:1 match with Figma (≤2px deviation)  
✅ **Responsive Design** - Fully responsive across all devices

## 🚀 Tech Stack

- **Next.js** (Page Router)
- **Tailwind CSS**
- **TypeScript** (optional but recommended)
- **DummyJSON API** for product data
